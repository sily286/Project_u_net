Authors:
- Wen Xu (Corresponding Author)

Description:
This package contains the source code, test data, and example outputs for the denoising method proposed in the above manuscript.  
All files are provided for reproducibility and academic use.

Files included:
- readme.txt (this file)
- Requirements.txt (user manual)
- dependence.txt
- Source code files (.py)
- Test data (small sample FIB-SEM images)
- Output examples (results generated by running the test data)

Please refer to Instructions.txt for detailed instructions on how to run the code.
